# Lead Mines Static Site

This project is now a static `HTML / CSS / JS` website.

## Run locally

1. Open `index.html` directly in your browser.
2. Or upload the project to GitHub Pages as-is.

## Main files

- `index.html`
- `blog.html`
- `courses.html`
- `giveaway.html`
- `contact.html`
- `assets/styles.css`
- `assets/app.js`

No backend is required.
